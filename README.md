# スマホテンキー (Numpad Remote)

スマートフォンをPCのテンキーとして使えるようにするツールです。
同じWi-Fiに接続しているだけで、スマホのブラウザから即座に使えます。

## 仕組み

```
スマホ (ブラウザ) ──WebSocket──▶ PC (numpad_server.py) ──PyAutoGUI──▶ キー入力
```

- **HTTPサーバー** (ポート `8766`): テンキーUIのHTMLをスマホに配信
- **WebSocketサーバー** (ポート `8765`): スマホからのキー入力を受信し、PyAutoGUIでPCに入力

## 必要環境

- Python 3.7+
- PC とスマホが同じ Wi-Fi に接続されていること

## インストール

```bash
pip install websockets pyautogui
```

## 使い方

1. PCでサーバーを起動する

```bash
python3 numpad_server.py
```

2. 起動時に表示されたURLをスマホのブラウザで開く

```
http://192.168.x.x:8766
```

3. テンキーが表示されたら使用開始

`Ctrl+C` でサーバーを終了できます。

## テンキーレイアウト

| CLR | ⌫ | / | × |
|:---:|:---:|:---:|:---:|
| 7 | 8 | 9 | - |
| 4 | 5 | 6 | + |
| 1 | 2 | 3 | Enter |
| 0 | 0 | . | Enter |

### キーの説明

| キー | 動作 |
|------|------|
| 0〜9, `.` | 数字・小数点を入力 |
| `+`, `-`, `×`, `/` | 演算子を入力 |
| `Enter` | Enterキーを送信 |
| `⌫` | Backspaceキーを送信 |
| `CLR` | 画面表示をクリア（PCへの入力なし） |

## ポート番号

| 用途 | ポート |
|------|--------|
| HTTP (UI配信) | 8766 |
| WebSocket | 8765 |

ファイアウォールでこれらのポートが許可されていない場合は設定を確認してください。
